# Multiagent gym environments

## Dependencies
* `gym`
* `numpy`

## Installation
To install package, use the following commands:
* `git clone https://github.com/cjm715/mgym.git`
* `cd mgym/`
* `pip install -e .`

## Environments
* TicTacToe-v0


## Testing
To run tests, install pytest with `pip install pytest` and run `python -m pytest`
